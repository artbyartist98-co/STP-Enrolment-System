Google Sheet Integration Steps (Minutes to Setup)
  1. Create a new Google Sheet and make sure the working sheet tab name is exactly Enrolments.
  2. In the Google Sheets menu, navigate to Extensions > Apps Script.
  3. Delete any dummy code present, and Paste the script code provided below.
  4. Click the Deploy > New deployment button.
  5. Select configuration type Web App. Set "Execute as" to Me and "Who has access" to Anyone (crucial for fetch access).
  6. Click Deploy, approve standard OAuth prompts, copy the generated Web App URL, and paste it into the Config box on the left!
